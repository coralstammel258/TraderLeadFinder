## 🔍 What is TraderLeadFinder?

TraderLeadFinder helps you spot traders looking for custom trading bots, AI tools, or developer support. It collects posts and messages from Reddit, Twitter, GitHub, and Telegram. Then it scores each lead by buying intent, using a proven BM25 method. Leads get classified into three groups: HOT, WARM, or COLD, so you can focus on the best opportunities first.

This tool works without needing any keys, accounts, or complicated setup. You can start exploring leads right away.

## ⚙ Understanding Lead Scores

TraderLeadFinder uses a simple scale:

- **HOT:** High chance the trader wants to buy or hire now. Prioritize these leads.
- **WARM:** Some interest shown but less urgent. Good to check regularly.
- **COLD:** Low readiness. Useful for long-term tracking.

The program updates scores as it scans new posts and messages.
